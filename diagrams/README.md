# Database Diagrams

This directory contains visual representations of the database architecture used in the Niagara Agriculture Database project.

---

## ER Diagram

The Entity Relationship (ER) diagram illustrates the structure of the database and the relationships between the different entities involved in the agricultural management system.

The diagram provides a high-level overview of:

- Tables and their attributes
- Primary key relationships
- Foreign key dependencies
- Data flow between entities

This visualization helps explain how the database is organized and how the different components interact within the system.
